## Outils & Méthodes

### Gestion de Projet
- **Suivi :** Notion (tableau de bord hebdomadaire)
- **Communication tuteur :** Point hebdomadaire (visio/présentiel)
- **Reporting :** Rapport bi-mensuel avec KPIs

### Stack Technique (déjà en place)
- Frontend : Next.js 14, React, TypeScript
- Backend : Supabase (PostgreSQL)
- Paiements : Stripe
- Analytics : Vercel Analytics, Sentry
- Design : Figma, brand-identity documenté

### Méthodologie
- **Sprints de 2 semaines** avec objectifs clairs
- **OKRs trimestriels** alignés sur les phases
- **Rétrospectives** fin de chaque phase
