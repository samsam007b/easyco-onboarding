# Izzico Pitch Deck - StartLab Build I

Pitch deck scrollable HTML pour Izzico.

## Déploiement

```bash
# Installer Vercel CLI si nécessaire
npm i -g vercel

# Déployer
vercel
```

## Contenu

- **14 slides** : Problème, Solution, Marché, Compétition, Produit, Business Model, Roadmap, Team
- **3 rôles** : Searchers, Residents, Owners avec leur value proposition détaillée
- **Design system v3-fun** : Gradients signature, couleurs de rôle, animations
