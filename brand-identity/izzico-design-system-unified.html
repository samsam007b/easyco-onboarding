Dark Mode - Warm Charcoal

Palette sombre avec tons chauds pour une expérience nocturne agréable

Philosophie Dark Mode

Le dark mode Izzico utilise une palette "Warm Charcoal" plutôt qu'un noir pur. Cette approche réduit la fatigue oculaire et maintient la chaleur de la marque.

Principe: Le dark mode utilise des gris chauds (#1A1A2E au lieu de #000000) pour éviter un contraste trop dur et préserver l'ambiance accueillante d'Izzico.

Accessibilité

WCAG AA + ΔE > 7 + Contrast Ratios

Règles d'Accessibilité

  • WCAG AA minimum: Ratio de contraste 4.5:1 pour tout texte
  • ΔE > 7: Différence perceptuelle entre couleurs distinctes (seuil de perception humaine)
  • Progression L* monotone: Luminosité décroissante de 50 (light) vers 900 (dark)
  • Variants TEXT: Owner 700, Resident 700, Searcher 700 sont text-safe (WCAG AA)
  • Pas de couleur seule: Utiliser aussi forme, texte, icône
Critère Standard Nos Couleurs
Contraste minimum WCAG AA 4.5:1 [OK] Searcher 700 sur 100 = 7.2:1
Texte sur gradient Minimum 4.5:1 sur tout le gradient [WARNING] À tester au cas par cas
Perception couleur ΔE > 7 entre couleurs distinctes [OK] Respecté dans les 3 rôles

Comment appliquer le Design System sur chaque plateforme

🌐 Web - Next.js + TailwindCSS

Structure des fichiers

app/ ├── globals.css # CSS variables (SOURCE DE VÉRITÉ) ├── fonts.ts # Déclaration des fonts Next.js └── layout.tsx # HTML root avec font classes lib/constants/ ├── searcher-theme.ts # Constantes JS Searcher ├── owner-theme.ts # Constantes JS Owner ├── resident-theme.ts # Constantes JS Resident └── README.md # Guide de synchronisation

Workflow de modification

  1. 1. Modifier dans app/globals.css (source)
    Exemple: --searcher-500: #ffa000;
  2. 2. Propager dans lib/constants/*-theme.ts (miroirs)
  3. 3. Tester visuellement - npm run dev
  4. 4. Commit avec message DESIGN SYSTEM
/* [OK] CORRECT - Utiliser CSS variables */ <button className="bg-searcher-500 text-white"> Search </button> /* ❌ INCORRECT - Hardcoder les couleurs */ <button className="bg-[#ffa000] text-white"> Search </button>
