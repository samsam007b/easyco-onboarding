## Pourquoi le Cercle ?

### Psychologie du Cercle

| Symbolisme | Application IzzIco |
|------------|-------------------|
| **Unite** | La colocation comme groupe uni |
| **Continuite** | Pas de debut ni de fin = communaute permanente |
| **Protection** | Safe space, foyer securisant |
| **Mouvement** | Cercle = rotation, vie qui tourne |
| **Inclusion** | Tout le monde peut entrer dans le cercle |

### Cercles Celebres

- **Spotify** : Cercle vert avec ondes sonores
- **Instagram** : Cercle avec gradient + camera
- **Mastercard** : Deux cercles qui s'intersectent
- **Target** : Cercles concentriques
- **TikTok** : Note dans forme arrondie
