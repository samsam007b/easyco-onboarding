Fusion des formes en un seul path vectorise

Probleme identifie

La version actuelle utilise des formes superposees (painter's algorithm). Les manches sont dessines SOUS le cercle, les caps SOUS le buste. Pour une vraie vectorisation, il faut fusionner ces formes en eliminant les parties cachees.

Version Originale Superposee

6 formes separees avec superposition. Les parties colorees sont CACHEES sous les blanches.

Version Clippee Propre

Manches clippes au bord du cercle. Pas de formes cachees.

Version Unifiee 1 Path

Toutes les formes dans un seul path compound avec fill-rule evenodd.
