Exploration des Icônes de Rôle

Loupe (Searcher) + Clé (Owner) + Personne (Resident) → Logo unifié ?

1. Les 3 icônes de rôle DESIGN SYSTEM

Searcher
Chercheur de logement
Cercle = verre de la loupe
Owner
Propriétaire
Cercle = tête de la clé
Resident
Résident actuel
Cercle = tête de la personne
Le point commun : le cercle
  • Les 3 icônes partagent un élément circulaire central
  • Ce cercle peut servir de point d'ancrage pour une fusion
  • Chaque "appendice" représente une fonction distincte

2. Anatomie : le cercle partagé

Loupe
Cercle + Manche diagonal
Clé
Cercle + Tige + Dents
Personne
Cercle (tête) + Corps

3. Concepts de fusion EXPLORATION

Concept A : Fusion centrale
Un cercle, trois extensions représentant chaque rôle
Concept B : Venn Diagram
Trois cercles = trois rôles, intersection = communauté
Concept C : Cercle englobant
Un cercle contenant les 3 symboles miniatures
Concept D : Abstrait tripode
Un cercle avec 3 rayons = 3 directions, 1 communauté

5. Concept hybride : le "i" transformé NOUVEAU

Et si le "i" de Fredoka pouvait évoquer les 3 rôles tout en restant simple ?

"i" standard
Le point = cercle = élément partagé des 3 icônes
"i" communautaire
Le point devient un trio = les 3 rôles unis
"i" accueillant
Le "i" devient clairement une personne qui accueille
"i" fusionné
Personne + éléments loupe/clé subtilement intégrés
Avantages du "i" transformé
  • Conserve la simplicité et la lisibilité à petite taille
  • Cohérence parfaite avec le wordmark "izzico"
  • Le point du "i" EST déjà un cercle = l'élément partagé
  • Peut évoluer subtilement sans perdre la reconnaissance
