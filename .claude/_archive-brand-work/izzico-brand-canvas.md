# Brand Canvas : IzzIco

> Document de reference pour le logo et l'identite visuelle.
> Genere le 2 janvier 2026 via le processus /logo-discover.

---

## Essence

### Origine du Nom
**IzzIco** = Easy + Co(living)
Rendre le coliving facile, accessible, humain.

### Mission
Reconnecter le "CO" dans coliving - transformer une recherche immobiliere froide en matching humain sincere.

### Vision (5 ans)
Eliminer le turnover cause par l'incompatibilite. Creer des liens durables, pas juste des baux.

### Phrase Cle
> "A quoi sert une grande cuisine si on ne se sent pas bien avec ceux qui y vivent ?"

### Le Probleme Resolu
- Les gens jugent le coliving comme de l'immobilier (m2, prix, localisation)
- Ils oublient le "CO" = cohabitation humaine
- Resultat : turnover, mal-etre, espaces communs vides dans des maisons pleines
- Les apps actuelles sont des marketplaces froides sans ame

## Public

### Marche Reel
Etudiants en premiere experience coliving
- IHECS, ULB, autres universites bruxelloises
- Budget serre, besoin de colocation
- Souvent dans l'urgence

### Persona Aspirationnel (Facade)
Jeunes professionnels 25-30 ans
- Early career, €2000-3500/mois
- Travaillent dans l'evenementiel, les bars branches, le creatif
- "Les gens un peu a la mode"
- Veulent du premium, sont habitues a Airbnb/Uber/Booking

### Les 3 Roles (USP)
1. **Searcher** (Jaune) - Cherche un coliving
2. **Resident** (Orange) - Vit deja en coliving
3. **Owner** (Violet) - Possede/gere une propriete

→ IzzIco est la seule app a couvrir les 3 roles (360°)

### Anti-Persona
"6 etudiants en premiere annee qui font des beuveries jusqu'a 5h du matin"
(Meme si c'est une partie du marche, ce n'est PAS l'image a projeter)

## Personnalite

### Metaphore
**Le connecteur cool a une soiree**
Decontracte mais efficace, te branche avec les bonnes personnes sans forcer.

### Ton
**Fun / Dynamique** (comme Spotify, Tinder)
Pas serieux-corporate, pas cheap-desorganise.

### 6 Adjectifs
- Fiable
- Humain
- Connecte
- Facile
- Rapide
- Fun

### 3 Anti-Adjectifs
- ✗ Cheap (pas low-cost, pas Appartager)
- ✗ Exclusif (pas snob, pas reserve a une elite)
- ✗ Froid (pas corporate, pas agence immo)

### Tension a Equilibrer
- Premium (qualite Airbnb) MAIS accessible
- Fun/dynamique MAIS fiable
- Humain/connecte MAIS rapide/efficace

## Differenciation

### USP #1 : Onboarding Dating-App-Style
Questionnaire de persona comme Tinder/Bumble.
"Personne ne fait ca hors des apps de rencontres."
C'est le game-changer du marche.

### USP #2 : Le 3eme Role (Residents)
Tous les autres : Searcher ↔ Owner
IzzIco : Searcher ↔ **Resident** ↔ Owner

Les residents etaient completement oublies du marche.

### USP #3 : Outils du Quotidien
- **Pour Owners** : Gestion propriete (pas juste annonces)
- **Pour Residents** : Courses, factures, wifi, regles, taches
- "Gerer la vie de 8 personnes qui vivent ensemble"

### Anti-Visuel (Ce que font les concurrents)
- Design date/cheap (Appartager, Room Lala)
- Froideur immobiliere (photos d'appart sans ame)
- Pas de travail sur les couleurs
- Aucun onboarding digne de ce nom

### Bonnes References
- Housing Anywhere : couleurs chaudes, personnages stylises, identite
- Coabs : premium/Booking.com (mais plus froid, style hotel)

## Direction Visuelle

### Type de Logo
**Symbole + Wordmark** (comme Airbnb, Slack)
- Un symbole fort qui peut vivre seul
- Le nom "IzzIco" stylise

### Inspirations Citees
- Airbnb (Belo, appartenance)
- Tinder (matching, social)
- Spotify (energie, fun)
- Instagram (gradient social)
- Housing Anywhere (couleurs chaudes)
- Notion (clean, moderne)

### Strategie Couleur : Le Gradient

**Concept** : Les 3 roles = 3 couleurs fluides

| Role | Couleur | Signification |
|------|---------|---------------|
| Owner | Violet/Mauve | Premium, tech, performance |
| Resident | Orange | Chaleur, creativite, equipe |
| Searcher | Jaune | Optimisme, innovation, action |

**Avantages du gradient** :
- Differenciant (peu de marques font des gradients)
- Association Instagram = social, communaute (OK)
- Logique fonctionnelle dans l'app (role = couleur)

**Probleme identifie** :
La transition violet→orange passe par un rose-orange "bizarre" qui, isole dans l'interface Owner, ne fonctionne pas bien.

**Solution possible** :
- Garder le gradient comme signature
- Creer des palettes separees par role pour l'UI

### Ce que le Logo Doit Evoquer

| Concept | Forme | Couleur |
|---------|-------|---------|
| Connexion | Formes liees | Gradient fluide |
| Communaute | Cercles, courbes | Couleurs chaudes |
| Confiance | Equilibre | Pas trop sature |
| Fun/Dynamique | Mouvement | Couleurs vives |
| Premium | Simplicite | Raffinement |
| 3 Roles | Triple element | 3 couleurs |

### Ce que le Logo Doit Eviter

| A eviter | Pourquoi |
|----------|----------|
| Maison generique | Trop "agence immo" |
| 3D / Effets glass | Date (logo actuel) |
| Trop de details | Perd en petit |
| Angles agressifs | Froid, corporate |
| Typo incoherente | Logo actuel problematique |

## 4 Directions Exploratoires

### Direction A : "Le Lien"
Trois elements (personnes/formes) connectes.
Symbolise les 3 roles qui s'unissent.
Inspiration : Slack, Olympics.

### Direction B : "Le Foyer Humain"
Maison abstraite avec element humain/social.
Le "home" qui est plus que des murs.
Inspiration : Airbnb Belo.

### Direction C : "Le ZZ Connecteur"
Les deux Z de IzzIco comme pont/lien.
Easy (IZZ) → Connection (I-CO).
Inspiration : Notion, Stripe.

### Direction D : "Le Cercle Social"
Cercle/bulle avec dynamisme interne.
Communaute, appartenance, completude.
Inspiration : Spotify, Mastercard.
