# Hero Background Images

## Instructions

Placez les 5 images d'intérieur que vous avez sélectionnées dans ce dossier avec les noms suivants :

1. `interior-1.jpg` - Image du salon vintage avec décoration
2. `interior-2.jpg` - Image de la salle à manger moderne avec escalier
3. `interior-3.jpg` - Image du salon bohème avec plantes suspendues
4. `interior-4.jpg` - Image du bureau avec plante dracaena
5. `interior-5.jpg` - Image du salon moderne avec coussins colorés

## Spécifications Recommandées

- **Format** : JPG ou WebP
- **Dimensions** : 1920x1080px minimum (ratio 16:9)
- **Taille** : < 500KB par image (compression recommandée)
- **Qualité** : 80-85% pour un bon compromis qualité/performance

## Optimisation

Les images seront :
- Chargées avec Next.js Image (optimisation automatique)
- Affichées avec 12% d'opacité par défaut
- Défilées lentement de gauche à droite (45 secondes par cycle)
- Recouvertes d'un dégradé purple-50 → white → gray-50

## Alternative

Si vous ne disposez pas encore des images, vous pouvez utiliser des services comme :
- Unsplash.com (chercher "modern interior", "cozy living room")
- Pexels.com (chercher "apartment interior", "home decor")
- Pixabay.com (images libres de droits)

Téléchargez des images de qualité correspondant au style EasyCo (moderne, chaleureux, accueillant).
